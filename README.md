# Data Analytics Project

## Overview

This project demonstrates an end-to-end data analytics workflow, starting from data loading and cleaning to database analysis and dashboard creation. The goal is to transform raw data into meaningful business insights using industry-standard analytics tools.

The project includes:

* Data loading using Python
* Exploratory Data Analysis (EDA)
* Data cleaning and preprocessing
* SQL analysis using PostgreSQL, MySQL, and SQL Server
* Interactive dashboard development in Power BI
* Project report documentation
* Presentation creation using Gamma

---

## Dataset

The dataset contains structured business data used for analysis and visualization.

Key characteristics:

* Multiple records with numerical and categorical features
* Missing values and duplicate records handled during preprocessing
* Suitable for SQL analysis and dashboard reporting

## Project Workflow

### 1. Data Loading

* Imported the dataset into Python.
* Performed initial inspection of the data.

### 2. Exploratory Data Analysis (EDA)

* Checked dataset structure.
* Analyzed data types.
* Generated summary statistics.
* Explored feature distributions.
* Identified trends and relationships.

### 3. Data Cleaning

* Removed duplicate records.
* Handled missing values.
* Corrected inconsistent data.
* Converted data types where required.
* Prepared the dataset for analysis.

### 4. SQL Analysis

Executed analytical SQL queries using:

* PostgreSQL
* MySQL
* SQL Server

The analysis included:

* Filtering

* ORDER BY
* Joins
* Subqueries
* Business insight generation

### 5. Power BI Dashboard

Created an interactive dashboard featuring:

* KPIs

* Slicers and filters
* Trend analysis
* Business performance metrics

### 6. Report Creation

Prepared a detailed project report covering:

* Objectives
* Methodology
* Analysis
* Findings
* Conclusions

### 7. Presentation

Created a professional presentation using Gamma to summarize:

* Project objectives

## How to Run

1. Clone this repository.
2. Open the Python notebooks in Jupyter Notebook or VS Code.
3. Install the required Python libraries.
4. Load the dataset.
5. Run the notebooks in sequence:

   * Data Loading
   * EDA
   * Data Cleaning
6. Execute the SQL scripts in PostgreSQL, MySQL, or SQL Server.
7. Open the Power BI dashboard (`.pbix`) to explore the visualizations.
8. Review the project report and presentation for a complete overview.
